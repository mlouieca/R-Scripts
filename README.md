# R-Scripts

Utility R and Power Query scripts for data work used across labour market, Census/CPS, and Power BI analysis projects.

## Contents

- R setup scripts for installing common analysis packages.
- CPS monthly summary scripts for U.S. labour-market data.
- Power Query unzip parsers for working with StatCan and other ZIP-based data sources, including ZIP64-compatible variants.

## Notes

These scripts are maintained as practical analysis helpers. Review script parameters and source URLs before running them in a new environment.
